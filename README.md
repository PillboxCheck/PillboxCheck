# PillboxCheck

**Offline AI Assistant for Medication Management using IBM Granite**

PillboxCheck is a multimodal, offline AI-powered desktop application designed to assist elderly individuals and others managing complex medication schedules. It utilizes IBM's Granite 3.2 large language model and integrates voice, OCR, and scheduling to improve adherence, prevent errors, and support patient autonomy â€” all while keeping user data entirely on-device.
